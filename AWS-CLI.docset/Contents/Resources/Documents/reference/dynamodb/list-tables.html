[ aws . dynamodb ]

list-tables¶

Description¶

Returns an array of all the tables associated with the current account and endpoint.

Synopsis¶

  list-tables
[--exclusive-start-table-name <value>]
[--limit <value>]

Options¶

--exclusive-start-table-name (string)

The name of the table that starts the list. If you already ran a list-tables operation and received a LastEvaluatedTableName value in the response, use that value here to continue the list.

--limit (integer)

A maximum number of table names to return.
