list-groups-for-user¶

Description¶

Lists the groups the specified user belongs to.

You can paginate the results using the --max-items and --marker parameters.

Synopsis¶

  list-groups-for-user
--user-name <value>
[--max-items <value>]
[--starting-token <value>]

Options¶

--user-name (string)

The name of the user to list groups for.

--max-items (integer)

The total number of items to return. If the total number of items available is more than the value specified in max-items then a NextMarker will be provided in the output that you can use to resume pagination.

--starting-token (string)

A token to specify where to start paginating. This is the NextToken from a previously truncated response.

Examples¶

To list the groups that an IAM user belongs to

The following list-groups-for-user commmand dislpays the groups that the IAM user named Bob belongs to:

aws iam list-groups-for-user --user-name Bob
