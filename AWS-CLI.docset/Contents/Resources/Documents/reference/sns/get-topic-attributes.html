get-topic-attributes¶

Description¶

The get-topic-attributes action returns all of the properties of a topic. Topic properties returned might differ based on the authorization of the user.

Synopsis¶

  get-topic-attributes
--topic-arn <value>

Options¶

--topic-arn (string)

The ARN of the topic whose properties you want to get.
