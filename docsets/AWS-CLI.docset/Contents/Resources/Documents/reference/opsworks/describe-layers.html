describe-layers¶

Description¶

Requests a description of one or more layers in a specified stack.

Synopsis¶

  describe-layers
--stack-id <value>
[--layer-ids <value>]

Options¶

--stack-id (string)

The stack ID.

--layer-ids (list)

An array of layer IDs that specify the layers to be described. If you omit this parameter, describe-layers returns a description of every layer in the specified stack.

Syntax:

"string" "string" ...

Examples¶

To describe a stack's layers

The following describe-layers commmand describes the layers in a stack, whose ID is 38ee91e2-abdc-4208-a107-0b7168b3cc7a:

aws opsworks describe-layers --stack-id 38ee91e2-abdc-4208-a107-0b7168b3cc7a
