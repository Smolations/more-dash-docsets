[ aws . opsworks ]

set-permission¶

Description¶

Specifies a stack's permissions. For more information, see Security and Permissions .

Synopsis¶

  set-permission
--stack-id <value>
--iam-user-arn <value>
[--allow-ssh | --no-allow-ssh]
[--allow-sudo | --no-allow-sudo]

Options¶

--stack-id (string)

The stack ID.

--iam-user-arn (string)

The user's IAM ARN.

--allow-ssh | --no-allow-ssh (boolean)

The user is allowed to use SSH to communicate with the instance.

--allow-sudo | --no-allow-sudo (boolean)

The user is allowed to use sudo to elevate privileges.
