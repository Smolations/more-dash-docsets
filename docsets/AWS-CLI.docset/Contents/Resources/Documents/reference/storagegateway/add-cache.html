[ aws . storagegateway ]

add-cache¶

Description¶

This operation configures one or more gateway local disks as cache for a cached-volume gateway. This operation is supported only for the gateway-cached volume architecture (see Storage Gateway Concepts ).

In the request, you specify the gateway Amazon Resource Name (ARN) to which you want to add cache, and one or more disk IDs that you want to configure as cache.

Synopsis¶

  add-cache
--gateway-arn <value>
--disk-ids <value>

Options¶

--gateway-arn (string)

The Amazon Resource Name (ARN) of the gateway. Use the list-gateways operation to return a list of gateways for your account and region.

--disk-ids (list)

An array of strings that identify disks that are to be configured as cache. Each string in the array must be minimum length of 1 and maximum length of 300. You can get the disk IDs from the list-local-disks API.

Syntax:

"string" "string" ...
