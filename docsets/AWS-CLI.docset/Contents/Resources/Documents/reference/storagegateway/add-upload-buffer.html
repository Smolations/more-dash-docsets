[ aws . storagegateway ]

add-upload-buffer¶

Description¶

This operation configures one or more gateway local disks as upload buffer for a specified gateway. This operation is supported for both the gateway-stored and gateway-cached volume architectures.

In the request, you specify the gateway Amazon Resource Name (ARN) to which you want to add upload buffer, and one or more disk IDs that you want to configure as upload buffer.

Synopsis¶

  add-upload-buffer
--gateway-arn <value>
--disk-ids <value>

Options¶

--gateway-arn (string)

The Amazon Resource Name (ARN) of the gateway. Use the list-gateways operation to return a list of gateways for your account and region.

--disk-ids (list)

An array of strings that identify disks that are to be configured as upload buffer. Each string in the array must be minimum length of 1 and maximum length of 300. You can get disk IDs from the list-local-disks API.

Syntax:

"string" "string" ...
