[ aws . storagegateway ]

describe-cachedi-scsi-volumes¶

Description¶

This operation returns a description of the gateway volumes specified in the request. This operation is supported only for the gateway-cached volume architecture.

The list of gateway volumes in the request must be from one gateway. In the response Amazon Storage Gateway returns volume information sorted by volume Amazon Resource Name (ARN).

Synopsis¶

  describe-cachedi-scsi-volumes
--volume-ar-ns <value>

Options¶

--volume-ar-ns (list)

An array of strings, where each string represents the Amazon Resource Name (ARN) of a cached volume. All of the specified cached volumes must be from the same gateway. Use list-volumes to get volume ARNs of a gateway.

Syntax:

"string" "string" ...
