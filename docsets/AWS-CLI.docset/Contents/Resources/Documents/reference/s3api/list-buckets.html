[ aws . s3api ]

list-buckets¶

Description¶

Returns a list of all buckets owned by the authenticated sender of the request.

Synopsis¶

list-buckets

Options¶

None

Examples¶

List Your Buckets¶

The following example uses the list-buckets command to display the names of all your Amazon S3 buckets (across all regions):

aws s3api list-buckets | jq ".Buckets[].Name"

The example uses jq to filter the output of list-buckets down to only the bucket names.

For more information about buckets, see Working with Amazon S3 Buckets in the Amazon S3 Developer Guide.
