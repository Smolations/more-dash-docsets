change-message-visibility-batch¶

Description¶

This is a batch version of change-message-visibility . It takes multiple receipt handles and performs the operation on each of the them. The result of the operation on each message is reported individually in the response.

Synopsis¶

  change-message-visibility-batch
--queue-url <value>
--entries <value>

Options¶

--queue-url (string)

The URL of the SQS queue to take action on.

--entries (list)

A list of receipt handles of the messages for which the visibility timeout must be changed.

Shorthand Syntax:

  Key value pairs, with multiple values separated by a space.
--entries Id=string,ReceiptHandle=string,VisibilityTimeout=integer

JSON Syntax:

[
  {
    "Id": "string",
    "ReceiptHandle": "string",
    "VisibilityTimeout": integer
  }
  ...
]
